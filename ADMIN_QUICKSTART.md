## 📝 Как добавить запись:

### Пример 1: Добавить данные здоровья

1. Откройте вкладку "💪 Здоровье"
2. Нажмите "➕ Добавить запись"
3. Заполните поля:
   - **ID** - `Автоматически` (disabled) ✅
   - **user_id** - Выберите `909006016: Иван` ИЛИ введите `123456789` ✅
   - **imt** - введите `25.5`
   - **imt_str** - введите `Нормальный`
   - **cal** - введите `2000`
   - **date** - `2025-11-08` (автоматически)
   - **weight** - введите `75.5`
   - **height** - введите `180`
4. Нажмите "✅ Сохранить"

✅ **Результат:** Запись добавлена, ID сгенерирован автоматически!

---

### Пример 2: Добавить питание для нового пользователя

4. Сохранить

✅ **Результат:** Запись добавлена для любого Telegram ID!
