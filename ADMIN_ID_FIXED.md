### 1. Функция `is_auto_column` полностью переписана

**Новая логика (простая и понятная):**

```python
def is_auto_column(table_name, column_name):
    # Timestamps всегда автоматические
    if column_name in ['created_at', 'updated_at', 'last_login']:
        return True
    
    # user_id в user_main НЕ автоматический (это Telegram ID)
    if table_name == 'user_main' and column_name == 'user_id':
        return False
    
    # user_id в user_aims НЕ автоматический (первичный ключ, но вводится)
    if table_name == 'user_aims' and column_name == 'user_id':
        return False
    
    # user_id в user_lang НЕ автоматический
    if table_name == 'user_lang' and column_name == 'user_id':
        return False
    
    # user_id в water НЕ автоматический
    if table_name == 'water' and column_name == 'user_id':
        return False
    
    # 'id' во ВСЕХ остальных таблицах = АВТОМАТИЧЕСКИ
    if column_name == 'id':
        return True
    
    return False
```
